You must have heard the buzz about **Windows 8**. If you haven’t, it is the newest OS by Microsoft, aiming to make a ‘**personalized home screen**‘ Somewhat like a mobile. This ‘**merging**‘ of Desktop OS and Mobile OS have made some fans upset, while some hopeful. Here is a **Official Video trailer**. 
  


  

What About You? You like it or Not?
  

If you liked it, You will be Glad to know that Microsoft has released its **Developer Build**, **Officially**. If you were scanning torrents daily to get a correct, working build of Windows 8, wait no more, and [**Download Windows 8 Dev Version**](http://msdn.microsoft.com/en-us/windows/home)
  

As Microsoft has said, **this version would nor be stable neither would contain all the features of the Retail version**, but it is made to give developers of Apps and Software to know what they would be working with, and give them some time for Ideas. 
  

If you want to wait for the stable version, it will hit the markets sometime next year.
